# v0.5.0

## Breaking Changes

- The recommended navigation format now moves from legacy `sections` to `collections`.
- Title fallback behavior in the export toolbar was adjusted for untitled pages.

## Added

- Added a more consistent metadata format for exported content.
- Improved prev/next page resolution.

## Migration

If you still use the legacy sidebar structure, migrate gradually to `collections`.
