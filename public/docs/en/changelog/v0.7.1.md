# v0.7.1

## Added

- Added extended Markdown image options for `width`, `height`, `size`, `preview`, `inline`, `x`, and `y`.
- Added support for native `<img>` controls via `data-preview`, `data-inline`, `data-offset-x`, and `data-offset-y`.
- Added inline rendering and offset controls for small icons, making text-baseline alignment and fine positioning easier.

## Fixed

- Fixed the issue where small images or SVG icons were stretched into full-width blocks when the image viewer was enabled.

## Docs

- Added the extended image syntax to the formal guide documentation.
- Synced the Chinese and English test pages, plus the matching docs in the `create-react-docs-ui` template.
