# v0.7.5

## Fixed

- Fixed `generate-doc-git-meta` so it can resolve commit metadata when the docs app lives inside a git subdirectory.
- Fixed `generate-changelog-index`, `generate-llms-files`, and `generate-search-index` to handle Windows `CRLF` frontmatter reliably.
- Fixed changelog summaries, last-updated metadata, and LLM exports incorrectly falling back to raw frontmatter text.

## Engineering

- Unified script handling for `LF`, `CRLF`, and BOM-prefixed content to keep parsing behavior consistent across generators.
