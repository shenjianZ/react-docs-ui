# v0.7.6

## Added

- Added centralized `react-docs-ui` CLI entry point to manage `generate-sitemap`, `generate-feed`, `generate-search-index`, `generate-doc-git-meta`, `generate-changelog-index`, `generate-llms-files` and other build scripts.
- Added local development workflow documentation, allowing `create-react-docs-ui` templates to reference local `react-docs-ui` source via `file:` protocol for joint development.

## Fixed

- Fixed frontmatter parsing failure for Markdown documents with Windows `CRLF` line endings.
- Fixed custom MDX components (Alert, Feature, Callout, etc.) not being recognized in template projects.

## Engineering

- Consolidated all build scripts under the `react-docs-ui` CLI; templates no longer need to reference script paths directly.
- Simplified `predev` / `prebuild` lifecycle hooks to call `react-docs-ui generate-*` subcommands.
