# v0.8.0

## Added

- **Sitemap Generation**: Automatically generates `sitemap.xml` at build time by scanning all doc pages. Supports `changefreq`, `priority`, and `exclude` config options; filters out `draft` and `noindex` pages.
- **RSS Feed**: Generates RSS 2.0 `feed.xml` from changelog indexes at build time. Browser auto-discovery via `<link rel="alternate">`. Supports `title`, `description`, and `limit` config options.
- **Breadcrumb Navigation**: Automatically generates hierarchical navigation paths based on sidebar config. Supports `breadcrumb.enabled` and `breadcrumb.showHome` config toggles.
- **Reading Time Estimation**: Automatically calculates estimated reading time for Chinese and English content. Supports `reading.showTime` config (default: enabled).
- **Reading Progress Bar**: Displays scroll-based reading progress at the top of the page. Supports `reading.showProgress` config (default: disabled).

## Security Fixes

- Fixed XSS risk in `MdxContent.tsx` where `dangerouslySetInnerHTML` rendered code block HTML. Added `sanitizeHtml()` to strip dangerous tags and event handlers.
- Fixed SVG injection risk in `Mermaid.tsx` caused by `securityLevel: "loose"`. Changed to `"strict"` mode.
- Fixed HTML injection risk in `SearchItem.tsx` where search highlights inserted raw HTML. Switched to safe React component rendering.
- Unified clipboard copy logic, replacing deprecated `document.execCommand('copy')` with `navigator.clipboard` API (with fallback).

## Fixed

- Fixed Sitemap generating backslash paths on Windows, causing malformed URLs.
- Fixed Sitemap `priority` outputting integers instead of decimal format (e.g. `1` → `1.0`).
- Fixed component loading failures being silent. Now shows toast error notifications.
- Fixed ChangelogPage silently returning empty data on config load failure. Added error UI.
- Fixed breadcrumb title duplication with page title.

## Engineering

- Added `src/lib/utils.ts` shared utility functions (`copyToClipboard`, `sanitizeHtml`).
- Added `src/lib/reading-time.ts` reading time estimation module.
- Added `src/components/Breadcrumb.tsx` and `ReadingProgressBar.tsx` components.
- `site.yaml` now supports `sitemap`, `feed`, `breadcrumb`, and `reading` config sections.
