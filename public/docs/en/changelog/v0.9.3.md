# v0.9.3

## Features

- Added a fullscreen viewer for Mermaid diagrams: click the fullscreen button in the top-right corner of a diagram to open it in a dialog with mouse-wheel zoom, drag-to-pan, and pinch-to-zoom.
- Viewer toolbar: zoom in, zoom out, fit, actual size, reset, download SVG, and native browser fullscreen, plus keyboard shortcuts `+` `-` `0` `F`.
- The Mermaid component now accepts `lang` and `viewerConfig` props; a new `MermaidViewerConfig` lets you customize viewer labels from site config.

## Bug Fixes

- Disabled `useMaxWidth` for all Mermaid diagram types and stripped the inline `max-width` from rendered SVGs, so large diagrams are no longer scaled down to the container width and their text stays readable.

## Engineering

- Bumped the package version to `0.9.3`.
