# Configuration

Config file: `public/config/site.yaml`

## site

Basic site information.

| Field | Description |
|-------|-------------|
| title | Site title |
| description | Site description |
| logo | Logo (emoji or image path) |
| author | Author |

## navbar

Top navigation bar.

| Field | Description |
|-------|-------------|
| showLogo | Show logo |
| showTitle | Show title |
| showLanguageSwitcher | Show language switcher |
| items | Navigation items |
| actions | Action buttons |

## sidebar

Sidebar navigation.

| Field | Description |
|-------|-------------|
| enabled | Enable sidebar |
| collections | Navigation collections |

## theme

Theme configuration.

| Field | Description |
|-------|-------------|
| defaultMode | Default mode (light/dark/auto) |
| allowToggle | Allow theme toggle |

## toc

Table of contents configuration.

| Field | Description |
|-------|-------------|
| enabled | Enable TOC |
| maxLevel | Max heading level |
| title | TOC title |

## footer

Footer configuration.

| Field | Description |
|-------|-------------|
| enabled | Enable footer |
| copyright | Copyright text |
| links | Link list |
| social | Social links |

## pwa

PWA configuration.

| Field | Description |
|-------|-------------|
| enabled | Enable PWA |
| name | App name |
| shortName | Short name |

## contextMenu

Context menu configuration.

| Field | Description |
|-------|-------------|
| enabled | Enable context menu |
| page | Page menu items |
| site | Site menu items |
| appearance | Appearance menu items |
