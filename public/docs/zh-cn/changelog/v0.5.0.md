# v0.5.0

## 破坏性变更

- 导航配置推荐从旧版单一 `sections` 结构迁移到 `collections` 结构。
- 导出工具栏在无标题页面中的标题回退逻辑已调整。

## 新增

- 为导出能力补充更一致的元信息格式。
- 为页面导航补充更明确的上一页 / 下一页判断。

## 迁移建议

如果你仍在使用旧版侧边栏结构，建议逐步迁移到 `collections` 配置。
