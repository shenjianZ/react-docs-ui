# v0.7.1

## 新增

- 为 Markdown 图片增加扩展参数，支持 `width`、`height`、`size`、`preview`、`inline`、`x`、`y`。
- 支持在原生 `<img>` 标签上使用 `data-preview`、`data-inline`、`data-offset-x`、`data-offset-y` 控制图片行为。
- 为小图标场景补充行内渲染与偏移能力，便于对齐文字基线和微调位置。

## 修复

- 修复开启图片预览器时，小尺寸图片或 SVG 图标被强制拉伸为整行大图的问题。

## 文档

- 补充正式 guide 文档中的图片扩展语法说明。
- 同步更新中英文测试页，以及 `create-react-docs-ui` 模板中的对应文档示例。
