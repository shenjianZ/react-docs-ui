# v0.7.5

## 修复

- 修复 `generate-doc-git-meta` 在 docs 项目位于 git 子目录时无法正确读取提交时间和作者的问题。
- 修复 `generate-changelog-index`、`generate-llms-files`、`generate-search-index` 对 Windows `CRLF` 文档 frontmatter 解析不稳定的问题。
- 修复 changelog 摘要、页面最近更新时间和 LLM 导出内容可能错误包含 frontmatter 原文的问题。

## 工程

- 统一构建脚本对 `LF`、`CRLF` 和 BOM 文本头部的处理逻辑，减少不同脚本之间的解析差异。
