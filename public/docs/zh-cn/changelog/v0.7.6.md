# v0.7.6

## 新增

- 新增 `react-docs-ui` 集中化 CLI 入口，统一管理 `generate-sitemap`、`generate-feed`、`generate-search-index`、`generate-doc-git-meta`、`generate-changelog-index`、`generate-llms-files` 等构建脚本。
- 新增本地联调工作流文档，支持 `create-react-docs-ui` 模板通过 `file:` 协议引用本地 `react-docs-ui` 源码进行联合开发。

## 修复

- 修复 Windows 环境下 `CRLF` 换行的 Markdown 文档 frontmatter 无法正确剥离的问题。
- 修复模板项目中自定义 MDX 组件（Alert、Feature、Callout 等）无法被识别的问题。

## 工程

- 将各构建脚本统一注册到 `react-docs-ui` CLI，模板项目不再需要单独引入脚本路径。
- `predev` / `prebuild` 生命周期钩子简化为调用 `react-docs-ui generate-*` 子命令。
