# v0.8.0

## 新增

- **Sitemap 自动生成**：构建时扫描所有文档页面自动生成 `sitemap.xml`，支持 `changefreq`、`priority`、`exclude` 配置，自动过滤 `draft` 和 `noindex` 页面。
- **RSS Feed**：构建时从 changelog 索引生成 RSS 2.0 `feed.xml`，浏览器自动发现（`<link rel="alternate">`），支持 `title`、`description`、`limit` 配置。
- **面包屑导航**：基于侧边栏配置自动生成层级导航路径，支持 `breadcrumb.enabled`、`breadcrumb.showHome` 配置开关。
- **阅读时间估算**：根据中英文内容自动计算预估阅读时间，支持 `reading.showTime` 配置（默认开启）。
- **阅读进度条**：页面顶部显示滚动阅读进度，支持 `reading.showProgress` 配置（默认关闭）。

## 安全修复

- 修复 `MdxContent.tsx` 中 `dangerouslySetInnerHTML` 渲染代码块 HTML 的 XSS 风险，新增 `sanitizeHtml()` 函数过滤危险标签和事件处理器。
- 修复 `Mermaid.tsx` 中 `securityLevel: "loose"` 导致的 SVG 注入风险，改为 `"strict"` 模式。
- 修复 `SearchItem.tsx` 搜索结果高亮直接插入 HTML 的注入风险，改用 React 组件安全渲染。
- 统一剪贴板复制逻辑，替换 `document.execCommand('copy')` 为 `navigator.clipboard` API（含降级方案）。

## 修复

- 修复 Sitemap 在 Windows 下生成路径使用反斜杠导致 URL 格式错误的问题。
- 修复 Sitemap `priority` 输出为整数而非规范的小数格式（如 `1` → `1.0`）的问题。
- 修复组件加载失败时用户无感知的问题，改为 toast 错误通知。
- 修复 ChangelogPage 配置加载失败时静默返回空数据的问题，增加错误提示 UI。
- 修复面包屑标题与页面标题重复显示的问题。

## 工程

- 新增 `src/lib/utils.ts` 共享工具函数（`copyToClipboard`、`sanitizeHtml`）。
- 新增 `src/lib/reading-time.ts` 阅读时间估算模块。
- 新增 `src/components/Breadcrumb.tsx`、`ReadingProgressBar.tsx` 组件。
- `site.yaml` 新增 `sitemap`、`feed`、`breadcrumb`、`reading` 配置节。
