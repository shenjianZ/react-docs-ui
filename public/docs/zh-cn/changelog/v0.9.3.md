# v0.9.3

## 新增

- Mermaid 图表新增全屏查看器：点击图表右上角的「全屏」按钮，可在对话框中查看完整图表，支持滚轮缩放、按住拖拽平移、双指捏合缩放。
- 查看器工具栏：放大、缩小、适配窗口、原始比例、重置、下载 SVG、浏览器原生全屏，并支持键盘快捷键 `+` `-` `0` `F`。
- Mermaid 组件新增 `lang` 与 `viewerConfig` 配置项；新增 `MermaidViewerConfig`，可在站点配置中自定义查看器标签文案。

## 修复

- 关闭 Mermaid 各图类型的 `useMaxWidth`，并清除渲染产物中的内联 `max-width`，图表不再被等比压缩到容器宽度，大图文字恢复清晰可读。

## 工程

- 包版本升级到 `0.9.3`。
