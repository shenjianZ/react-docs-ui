# 介绍

React Docs UI 是一个配置驱动的文档网站构建工具。

## 核心特性

1. **配置驱动**: 通过 YAML 配置文件控制所有功能
2. **MDX 支持**: 在 Markdown 中使用 React 组件
3. **主题系统**: 支持明暗主题，自动跟随系统
4. **导航系统**: 顶部导航、侧边栏、目录
5. **国际化**: 支持多语言文档
6. **快捷功能**: 命令菜单（Cmd+K）、右键菜单
7. **代码高亮**: 集成 rehype-highlight
8. **数学公式**: 支持 KaTeX 渲染
9. **PWA 支持**: 可配置为渐进式 Web 应用
10. **AI 集成**: 可配置的 AI 助手功能（需配置 API 密钥）

## 适用场景

- 开源项目文档
- 产品手册
- 团队知识库
- 技术博客
- API 文档
