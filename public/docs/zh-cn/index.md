# React Docs UI

配置驱动的 React 文档网站构建工具。

## 核心特性

- **配置驱动**: 通过 `site.yaml` 定义网站，无需编写代码
- **MDX 支持**: 在 Markdown 中使用 React 组件
- **主题切换**: 支持明暗主题自动切换
- **命令菜单**: `Cmd+K` 快捷访问
- **右键菜单**: 可配置的上下文菜单
- **国际化**: 基于配置文件的多语言支持
- **代码高亮**: 集成 rehype-highlight
- **数学公式**: 支持 KaTeX
- **PWA 支持**: 可配置的渐进式 Web 应用
- **AI 集成**: 可配置的 AI 助手（需配置 API 密钥）

## 快速开始

```bash
npx create-react-docs-ui@latest my-docs
cd my-docs
npm install && npm run dev
```
